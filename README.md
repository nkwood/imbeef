# imbeef

IMBE WAVE file transcoding via AWS Kinesis streams & AWS S3.

## Create imbeef.properties
Copy `example-imbeef.properties` to `imbeef.properties` and modify as you see fit.

## Build
```
$ mvn package
```

## Run
```
$ java -jar target/imbeef-0.2.jar
```
